❓ How to get this file
  1. Go to velocityfleet.com and select Trip History.

  3. Select All Vehicles.
  4. Enter a Specific Date Range (previous Monday – Sunday).
  5. Select Download, then Excel.
  6. Upload that file here.

❓ How to get this file
  1. Go to velocityfleet.com and select Trip History.
  2. Select Driver Performance.
  3. Select All Drivers.
  4. Enter a Specific Date Range (previous Monday – Sunday).
  5. Select Download, then PDF.
  6. Upload that file here.
