2 · Import job history (archive)

Load your previous jobs from the exported spreadsheet. They go into a separate archive — searchable on the Job Archive page, but kept out of the live scheduler so daily work stays fast. Safe to re-run: jobs are matched by their MOS number, so nothing duplicates.

Do this on a computer (not a phone). The file is large — leave the tab open until it finishes.

3 · Import job photos & files

Pull the historical photos, signatures and PDFs in from the manifest produced by the browser harvester (workever-manifest.json). The server fetches each file straight from storage and attaches it to the matching job by MOS number — shown on the Job Archive page. Safe to re-run: files already brought in are skipped, so if it stops you just start it again and it carries on.

This moves a lot of data (tens of GB across ~80k files) — it can take a while. Keep this tab open; it resumes if interrupted.
